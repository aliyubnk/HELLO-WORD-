## Introduction
The **Land Sales Management System** is designed to help real estate businesses and individuals manage land properties, track sales, and maintain customer information.  

Developed as an undergraduate **Computer Science project**, it demonstrates practical skills in database management, user interface design, and system integration.

---

## Features
- Add, edit, and remove land properties  
- Track sales transactions and payment status  
- Maintain detailed customer records  
- Generate reports for sales and property management  
- User-friendly interface for administrators and agents
